# Community Creation Copy Notes

Member-facing copy uses direct Korean wording that identifies the field to fix and preserves draft context.

- Duplicate address: `이미 사용 중인 주소입니다.`
- Restricted words: `커뮤니티 이름 또는 소개에 사용할 수 없는 표현이 포함되어 있습니다.`
- Review pending: `운영자 검수 대기 중입니다. 승인 전에는 디렉터리와 검색에 노출되지 않습니다.`
- Successful launch: `커뮤니티가 개설되었습니다. 생성자가 커뮤니티장으로 지정되었습니다.`
- Media validation: `PNG, JPEG, WebP 형식과 5MB 이하 파일을 사용할 수 있습니다.`

Operator copy is concise and audit-focused: decision controls require a reason code and explanatory text for approval, rejection, and change requests.

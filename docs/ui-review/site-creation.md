# UI Review: CMS Site Creation

Reference direction: Material Dashboard Shadcn.

Implemented alignment:

- Persistent `w-60` left sidebar with active black stone gradient nav state.
- Light `bg-stone-50` workspace with a bordered white main page frame.
- Compact white cards with `rounded-lg`, subtle stone borders, and minimal shadows.
- Dashboard status cards, table rows, badges, progress bars, and validation issue tables follow the reference density.
- Primary CTAs use the required stone gradient treatment; secondary actions use subdued outlined buttons.
- CMS-specific visuals are dependency graphs, stage timelines, progress indicators, and document previews rather than marketing imagery.

Screens covered:

- Dashboard
- Create CMS Site
- Generation Plan Review
- Drafts
- Domain Catalog
- Validation Reports
- Audit Log
- Settings

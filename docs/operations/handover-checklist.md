# Handover Checklist

- Source structure reviewed: `backend/`, `frontend/`, `database/`, `docs/`.
- Environment values documented and transferred through approved secret handling.
- PostgreSQL schema and init behavior explained.
- Upload volume backup and restore tested.
- Admin and portal user journeys demonstrated.
- Test commands and smoke test script demonstrated.
- Known risks recorded: PDF conversion quality, local storage capacity, future auth integration.
